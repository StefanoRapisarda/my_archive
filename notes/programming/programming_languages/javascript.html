Javascript

Definitions

Javascript is the language of the web. [...] Different browsers have different JavaScript interpreters which execute JavaScript code. [1]

JavaScript is a scripting languange that is one of the three core languages used to develop websites. Whereas HTML and CSS give a website strucure and style, Javascript lets you add functionality and behaviors to your website, allowing your website's visitors to interact with content in many imaginative ways.
JavaScript is primarily a client-side language, meaning it runs on your computer within your browser. However, more recently the introduction of Node.js has allowed JavaScript to also execute code on servers. [2]

Facts

  • Javascript specifications are indicated by ES and a number (e.g. ES6). ES stands for ECMAScript, where ECMA is the European Computer Manufacturers Association.

Classes

You may often find online that classes have been implemented in JavaScript starting from ES6, however class definition has always been allowed in JavaScript, but ES6 introduced a new, more convenient, syntax using the class keyword.
Here I list the main things to remember about classes:
  • Classes are defined in this way: class ClassName {}. In JavaScript classes start with capital letters (like in python) and the curly braces contain the body of the class declaration;
  • Classes are initialized with the constructor method (constructor(args) {}). Inside the constructor, class properties are referred using the word this ( this.attr1 = 1;). Recently, a new syntax has been approved, you can write the properties (or fields) of the class at the beginning of the class body simply listing the name of the fields and their value (attr1 = 1;). Using this latter syntax you can also define variable that are accessible only inside the classe: #attr = 1. Remember that in both syntaxes, not variable declaration is necessary.
    The constractor function is equivalent to __init__() in python;
  • Methods are defined in this way: methodName(args) {}. Contrary to python (where underscored are uside in function names), in Javascript the name convetion for methods (and functions in general) is words separated by capital letters;
  • Inheritance is achieved with the word extends: class ClassName extends ParentClass {}. In order to inherit all the properties of the parent classe, you need to use super() in the constructor of the child class;
  • Static (or class) methods are methods invoked on the class constructor, while standard methods are invoked on a class instance. In order to define standard static methods you simply need to use the prefix static before the method;
  • !!! When you define functions in javascript, that piece of code is executed at the beginning of your script. Even if the class declaration looks similar to the function declaration, this is not true for classes, so classes must be defined before being used.
