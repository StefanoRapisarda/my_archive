Programming Recommendations

From PEP8

  • !!! comparison with None should always be done with is or is not, never with equality operators. Also, always use is not instead of not is and always use if x is not None instead of if x when testing whether a variable or argument that default to None was set to some othe value;
  • Use ''.startswith() and ''.endswith() instead of string slicing to check for prefixes ot suffixes;
  • Object comparisons should always use isinstance();
