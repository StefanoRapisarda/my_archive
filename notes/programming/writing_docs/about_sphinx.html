About Sphinx

What I've learned from the first website page

  • Sphinx was originally created for python documentation;
  • The output formats include PDF, latex, and epub among others. Apparently you can even write a book with Sphinx (check Packt);
  • It uses reStructuredText, a markup language;
  • You can use it with many themes like alabaster, classic, nature (pandas, cython, PyWavelets), etc;
  • Read the Docs is not a competitor of sphinx as I thought, but rather a place to host your documentation for free;

From getting started

  • DEF: Sphinx is a documentation generator, i.e. a tool that translates a set of plain text source files into various outout formats;

First steps

  • Install sphinx with python -m pip install sphinx;
  • Verify sphinx installation with sphinx-build --version;
  • From the command line, inside your selected folder, run sphinx-quickstart docs. This will ask you a series of questions (answer yes to the first one) and will create a new docs folder. Inside this folder, in particular, you will find that the source/index.rst file is the root document of the project, which serves as welcome page and contains root of the "table of contents tree" (toctree);
  • To export your documentation in html (or any other format) run the command sphinx.build -b html docs/source/ docs/build/html. After this first build, when you modify the .rst file you can simply run make html inside the docs folder;
